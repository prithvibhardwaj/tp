**GymOps is a CLI-first desktop application for gym management, helping you track trainers, members, and sessions with speed and precision.** While it has a GUI, most of the user interactions happen using a CLI (Command Line Interface).

* If you are interested in using GymOps, head over to the [_Quick Start_ section of the **User Guide**](UserGuide.html#quick-start).
* If you are interested about developing GymOps, the [**Developer Guide**](DeveloperGuide.html) is a good place to start.
